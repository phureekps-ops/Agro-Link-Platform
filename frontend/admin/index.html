พอร์ทัลผู้ดูแลระบบ (Platform Ops) — อนุมัติ/ปฏิเสธ KYC เกษตรกร, อนุมัติ/ปฏิเสธ KYB องค์กร, ตรวจสุขภาพระบบ

หมายเหตุ: สภาพแวดล้อมนี้ยังไม่มีระบบบัญชีผู้ดูแลรายบุคคล (ไม่มี MFA / SSO จริง) — จึงใช้รหัสผ่านกลางชุดเดียว (ตั้งค่าที่ ADMIN_PASSCODE ใน .env ฝั่ง backend) แทนการยืนยันตัวตนของผู้ดูแลแต่ละคน ระบบจริงต้องมีบัญชีผู้ดูแลรายบุคคลเพื่อให้ตรวจสอบย้อนกลับ (audit) ได้ว่าใครเป็นผู้ทำรายการ
เกษตรกร / ผู้ปล่อยกู้ / ผู้รับซื้อผลผลิต / ผู้ให้บริการเครื่องจักรกล / ผู้จำหน่ายปัจจัยการผลิต?
ไปที่พอร์ทัลเกษตรกร ไปที่พอร์ทัลผู้ปล่อยกู้ ไปที่พอร์ทัลผู้รับซื้อผลผลิต ไปที่พอร์ทัลผู้ให้บริการเครื่องจักรกล/ลานตาก ไปที่พอร์ทัลผู้จำหน่ายปัจจัยการผลิต
